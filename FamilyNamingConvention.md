# Family Naming Convention

This document describes the naming convention used for families in the Meinhardt tools, particularly for the Family Renamer tool.

## General Rules
- Family names should be descriptive and follow a consistent format.
- Use underscores to separate words.
- Include category, type, and size where applicable.

## Example Formats
- For MEP families: CATEGORY_TYPE_SIZE_VENDOR
  - e.g., DUCT_RECTANGULAR_600x400_ACME
- For structural: STRUCTURAL_ELEMENT_MATERIAL_SIZE
  - e.g., BEAM_STEEL_W12x50

## Specific Conventions
- Ducts: DUCT_SHAPE_SIZE
- Pipes: PIPE_MATERIAL_SIZE
- Fittings: FITTING_TYPE_SIZE

This convention ensures consistency across projects.
